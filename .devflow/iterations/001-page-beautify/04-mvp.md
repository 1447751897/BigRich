# MVP 范围与计划 — 迭代:页面美化

> 输入:`01-requirements`(PRD,done)。02-design/03-tech-stack 在 main 仍为骨架,故以 PRD §3A–§3F + 硬约束(纯 `public/`、不碰 engine/gateway、不回归 38 测试、去 IP、掷骰服务端权威)直接落地;长期结论归档时回流 baseline。
> 增量选型(对齐 D-101 口径):**不引入新依赖**,原生 CSS 动画 + 原生 Web Audio 合成音效(无素材文件)。

## 1. MVP 范围(本迭代已交付)
- ✅ **自创主题「环游都市」(去 IP)**:都市街区色带(翡翠湾/金沙滩/霓虹巷/紫藤台/蓝湾港/朝阳门)、建筑分级 emoji(🏠→🏙️)、角色棋子、机遇/命运/税务/拘留所等中性命名;货币用 🪙。进房 / 大厅 / 对局 / 结果页全流程统一美化。
- ✅ **一屏自适应(桌面 + 手机竖屏不滚动)**:`100dvh` 防地址栏抖动;棋盘 `aspect-ratio:1/1` + `width:min(100cqw,100cqh)` 等比缩放**不重排**(保留 8×8 结构);桌面两栏 + 顶部头像条,手机竖屏单列 + **底部固定操作条** + 强交互 **bottom-sheet**(交易/拍卖/地产管理);事件日志 `<details>` 可折叠。
- ✅ **位置可见**:棋子覆盖层(`.pawn-layer`),各玩家彩色+角色图标双重区分,**自己金色描边光晕 + "(你)"**;位置变化经 CSS `transition` **沿格平滑移动**。
- ✅ **掷骰动画**:点击掷骰后骰子滚动演出,**收到服务端 `DiceRolled` 立即定格到该点数**;前端只用 `Math.random` 做视觉滚动、绝不决定结果;动画关闭档直接定格。
- ✅ **音效**:Web Audio 振荡器合成(掷骰/移动/买地/收租/抽卡/入狱/破产/胜利/轮到你/点击),**静音 + 音量** localStorage 记忆,**首个用户手势内解锁 AudioContext**。
- ✅ **三档动画**(完整/精简/关闭)+ **`prefers-reduced-motion` 强制关闭** + **启动 1s FPS 探测 <40 自动降精简**。
- ✅ **主题静态 mock**:`public/theme-mock.svg`(`tools/gen-mock.mjs` 生成,配色/布局与前端一致),供客户过样。

## 2. 明确不做(本迭代)
- 多主题切换 / 横屏专门适配(竖屏优先);上传头像;微信小程序端。
- 复杂粒子/canvas 特效(纯 CSS + emoji 控成本,零素材文件)。
